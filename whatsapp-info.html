WhatsApp Integration Guide

About WhatsApp Integration

WeCloud uses the WhatsApp HTTP API (WAHA) to send notifications and invoices to customers. This integration allows you to communicate with customers through WhatsApp, which has higher open and response rates compared to SMS or email.

Limitations of Free Version

The free version of WAHA has certain limitations that affect how we can send invoices:

  • No File Attachments: The free version does not support sending PDF files, images, or other attachments.
  • Text Messages Only: You can only send text messages with the free version.
  • Limited Formatting: Only basic formatting like bold, italic, and bullet points is supported.

When attempting to send a PDF invoice via WhatsApp, you'll receive an error message stating: "The feature is available only in Plus version."

Solutions and Workarounds

We've implemented the following solutions to work around these limitations:

Text-Based Invoices

When sending invoices via WhatsApp, we format the invoice details as a text message with all the essential information.

Invoice Viewer

Customers can view their invoices online through our web-based invoice viewer, which shows a professional invoice layout.

PDF Downloads

Customers can download their invoices as PDF files directly from the invoice viewer page.

Print Option

The invoice viewer includes a print option for customers who need physical copies.

Upgrading to Plus Version

If you need to send PDF invoices directly via WhatsApp, you can upgrade to the Plus version of WAHA. Visit https://waha.devlike.pro/ for more information on pricing and features.

How to Upgrade

  1. Visit the WAHA website
  2. Purchase a Plus license
  3. Update your Docker container to use the Plus version
  4. Restart the WAHA server

After upgrading, the system will automatically detect the Plus version and enable PDF invoice sending.
